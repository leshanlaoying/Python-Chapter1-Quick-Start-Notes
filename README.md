# Python语言快速入门学习笔记 Python Quick Start Learning Notes

本仓库记录了本人在学习 Python 过程中的第一阶段学习成果，主要内容来源于《Python语言快速入门》一书的学习整理。

仓库采用 **Jupyter Notebook + Markdown笔记 + Python代码实践** 的形式，对 Python 基础知识进行系统梳理。

相比传统代码文件，本项目中的 Jupyter 笔记按照章节进行划分，每个知识点均可以独立运行，适合初学者边学习理论边进行代码实践。
